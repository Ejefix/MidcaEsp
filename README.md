# MIDKA — Embedded Intent-Based Smart Home Core (ESP32)

MIDKA — это система умного дома на базе ESP32, построенная вокруг единой концепции:

> Любое событие в системе является намерением (Intent).

---

# 📌 Основная идея архитектуры

В системе нет “команд” в классическом виде.

Есть только Intent:

- пользователь хочет действие
- устройство сообщает событие
- сенсор фиксирует состояние
- всё превращается в Intent

Дальше система сама решает:
- что выполнить
- когда выполнить
- с каким приоритетом
- и что конфликтует между собой

---

# 🧠 Core концепция (Intent Architecture)

Любой источник в системе — это генератор Intent:

- пользователь (через приложение)
- датчики
- кнопки
- устройства
- внутренние сценарии системы

Все они равноправны на уровне входа.
